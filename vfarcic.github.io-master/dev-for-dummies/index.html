Servers

A server is a computer or a device on a network that manages network resources

Servers

Bare-Metal

A single tenant server

Servers

Bare-Metal

Pros

  • Best performance
  • Physical segregation of resources
  • Avoids the noisy neighbor effect
  • Higher level of security

Servers

Bare-Metal

Cons

  • Expensive
  • Not very agile and elastic

Servers

Virtual Machines

Partitioning of a physical server into a number of small, virtual servers

Based on hypervisors running on bare-metal

Most companies run their applications in virtualized servers

Servers

Virtual Machines

Pros

  • Can be easily created and destroyed
  • Resources can be adjusted
  • Resource isolation between different VMs
  • Horizontal scalability

Servers

Virtual Machines

Cons

  • Lower performance than bare metal
  • Potential of the noisy neighbor effect
  • Inconsistent performance
  • Potential security concerns

Servers

Containers

On top of a physical server and its host OS

As small as a few MB

A collection of unchangeable images

Servers

Containers

Pros

  • Speed
  • Low resource usage
  • Immutability
  • Efficiency

Servers

Containers

Cons

  • Resource isolation
  • Security
